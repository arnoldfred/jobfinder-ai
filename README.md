# JobFinder AI v2.1

Projet Django de recherche d'offres d'emploi avec automatisation, intégration IA et pipeline de scraping.

## Structure
- jobfinder2/: application Django principale
- docmem/: documents de mémoire

## Installation
1. Créer un environnement virtuel
2. Installer les dépendances depuis jobfinder2/requirements.txt
3. Lancer les migrations et le serveur

## Développement
```bash
cd jobfinder2
python manage.py runserver
```
